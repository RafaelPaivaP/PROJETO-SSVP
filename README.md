SSVP – Sistema de Doações

Aplicação web desenvolvida para facilitar o cadastro e gerenciamento de doações destinadas à Sociedade São Vicente de Paulo (SSVP). O sistema permite registrar doações, visualizar itens cadastrados e acompanhar o status de cada solicitação, tudo em uma interface simples e responsiva.

Instalação
1. Clone o repositório:
git clone https://github.com/RafaelPaivaP/PROJETO-SSVP.git

2. Acesse a pasta do projeto:
cd ssvp

3. Instale as dependências:
npm install

Uso
Iniciar o servidor de desenvolvimento:
npm run dev


Após iniciar, acesse no navegador:

http://localhost:5173

Descrição do Projeto

O sistema permite:

Cadastrar novas doações

Listar todas as doações registradas

Atualizar o status das doações (Pendente → Em Coleta → Entregue)

Excluir registros

Visualizar informações de contato

Utilizar armazenamento local (localStorage)

A aplicação foi construída utilizando React, Vite, HTML, CSS e JavaScript, priorizando simplicidade, organização e responsividade.
